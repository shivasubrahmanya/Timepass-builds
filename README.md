# Nokia Snake Game

A classic Snake game implementation built with HTML5 Canvas, JavaScript, and CSS. This project recreates the nostalgic feel of the original Nokia snake game with a modern touch.

## Features

*   **Classic Gameplay**: Navigate the snake to eat food and grow longer.
*   **Responsive Design**: Playable on desktop and mobile devices.
*   **Controls**: Supports both keyboard arrow keys and on-screen buttons.
*   **Score Tracking**: Real-time score updates.
*   **Increasing Difficulty**: The game speed increases slightly as you collect more food.
*   **Retro Aesthetics**: styled with a nostalgic color palette reminiscent of early mobile displays.

## How to Play

1.  **Start the Game**: Click the "Start Game" button to begin.
2.  **Controls**:
    *   **Desktop**: Use the Arrow Keys (↑, ↓, ←, →) to control the snake's direction.
    *   **Mobile / Mouse**: Click the on-screen directional buttons.
3.  **Objective**: Eat the red food items to grow the snake and increase your score.
4.  **Game Over**: The game ends if the snake hits the walls or its own body.
5.  **Reset**: Click the "Reset" button to stop the current game and reset the state.

## Setup & Running

This is a static web application, so it requires no backend setup.

1.  **Clone or Download** the repository.
2.  **Open `index.html`** in any modern web browser.
    *   You can double-click the file to open it locally.
    *   Alternatively, use a local development server (like Live Server in VS Code) for the best experience.

## Project Structure

*   `index.html`: The main HTML structure containing the game canvas and controls.
*   `script.js`: Contains all the game logic, including the game loop, snake movement, collision detection, and drawing functions.
*   `style.css`: Handles the styling of the game container, canvas, and controls to give it that retro look.

## Technologies Used

*   HTML5
*   CSS3
*   JavaScript (ES6+)
